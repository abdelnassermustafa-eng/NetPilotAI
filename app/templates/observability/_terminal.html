The Terminal panel provides a safe, read-only view of operational command examples used for observability and diagnostics. No commands are executed from this interface, and no infrastructure changes are performed in the current version.

This view is intended to help operators understand how backend data is inspected and retrieved, while maintaining full audit visibility and operational safety.

netpilot@observability:~$

# Inspect existing VPCs without modification

aws ec2 describe-vpcs \
  --query 'Vpcs[*].VpcId' \
  --output table
